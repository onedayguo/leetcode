# 9001. Bubble Sort

- Difficulty: Easy
- Link: https://en.wikipedia.org/wiki/Bubble_sort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using the bubble sort algorithm.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [5,1,4,2,8]
Output: [1,2,4,5,8]

## Example 2

Input: nums = [3,3,2,1]
Output: [1,2,3,3]

## Constraints

- `0 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
