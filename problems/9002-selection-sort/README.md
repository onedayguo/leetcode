# 9002. Selection Sort

- Difficulty: Easy
- Link: https://en.wikipedia.org/wiki/Selection_sort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using the selection sort algorithm.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [64,25,12,22,11]
Output: [11,12,22,25,64]

## Example 2

Input: nums = [2,1,2,1]
Output: [1,1,2,2]

## Constraints

- `0 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
