# 9003. Insertion Sort

- Difficulty: Easy
- Link: https://en.wikipedia.org/wiki/Insertion_sort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using the insertion sort algorithm.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [12,11,13,5,6]
Output: [5,6,11,12,13]

## Example 2

Input: nums = [1]
Output: [1]

## Constraints

- `0 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
