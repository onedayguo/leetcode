# 9004. Shell Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Shellsort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using the shell sort algorithm.

Use a gap-based insertion strategy. The gap sequence is up to you.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [12,34,54,2,3]
Output: [2,3,12,34,54]

## Example 2

Input: nums = [5,1,1,2,0,0]
Output: [0,0,1,1,2,5]

## Constraints

- `0 <= nums.length <= 2 * 10^4`
- `-10^9 <= nums[i] <= 10^9`
