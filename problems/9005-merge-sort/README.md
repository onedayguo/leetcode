# 9005. Merge Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Merge_sort

## Description

Given an integer array `nums`, sort the array in ascending order using merge sort.

Implement `mergeSort(nums []int)` and make sure `nums` is sorted after the call.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [38,27,43,3,9,82,10]
Output: [3,9,10,27,38,43,82]

## Example 2

Input: nums = [4,1,3,2]
Output: [1,2,3,4]

## Constraints

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
