# 9006. Quick Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Quicksort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using quick sort.

Choose any partition strategy, but do not call built-in sorting utilities.

## Example 1

Input: nums = [10,7,8,9,1,5]
Output: [1,5,7,8,9,10]

## Example 2

Input: nums = [3,3,3,2,2,1]
Output: [1,2,2,3,3,3]

## Constraints

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
