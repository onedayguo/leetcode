# 9007. Heap Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Heapsort

## Description

Given an integer array `nums`, sort the array in ascending order **in-place** using heap sort.

Build a heap manually and do not call any built-in sorting helper.

## Example 1

Input: nums = [12,11,13,5,6,7]
Output: [5,6,7,11,12,13]

## Example 2

Input: nums = [4,10,3,5,1]
Output: [1,3,4,5,10]

## Constraints

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
