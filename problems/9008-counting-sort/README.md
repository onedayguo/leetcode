# 9008. Counting Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Counting_sort

## Description

Given a **non-negative** integer array `nums`, sort the array in ascending order using counting sort.

You may use extra space for frequency counting, but do not use built-in sorting functions.

## Example 1

Input: nums = [4,2,2,8,3,3,1]
Output: [1,2,2,3,3,4,8]

## Example 2

Input: nums = [0,5,2,0,1]
Output: [0,0,1,2,5]

## Constraints

- `0 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^5`
