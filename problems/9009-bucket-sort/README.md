# 9009. Bucket Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Bucket_sort

## Description

Given a **non-negative** integer array `nums`, sort the array in ascending order using bucket sort.

You can choose the bucket size and the method used to sort values inside each bucket.

You are not allowed to call built-in sorting for the whole array.

## Example 1

Input: nums = [29,25,3,49,9,37,21,43]
Output: [3,9,21,25,29,37,43,49]

## Example 2

Input: nums = [5,1,4,2,8]
Output: [1,2,4,5,8]

## Constraints

- `0 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^6`
