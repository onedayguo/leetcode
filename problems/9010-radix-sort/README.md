# 9010. Radix Sort

- Difficulty: Medium
- Link: https://en.wikipedia.org/wiki/Radix_sort

## Description

Given a **non-negative** integer array `nums`, sort the array in ascending order using LSD radix sort (base 10).

You may use counting by digit for each pass.

You are not allowed to use any built-in sorting function.

## Example 1

Input: nums = [170,45,75,90,802,24,2,66]
Output: [2,24,45,66,75,90,170,802]

## Example 2

Input: nums = [5,3,1,0]
Output: [0,1,3,5]

## Constraints

- `0 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^9`
